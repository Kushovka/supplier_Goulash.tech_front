# Frontend (React)

MVP внутреннего сервиса для закупщика: поиск поставщиков продуктов питания,
карточки, рейтинг, сравнение и AI-анализ по кнопке.

## Что умеет сервис

- фильтровать поставщиков по категории товара;
- учитывать город или регион;
- показывать цену, минимальный заказ, документы, доставку и контакты;
- сравнивать выбранных поставщиков в таблице;
- помогать принять решение через рейтинг 0-10;
- запускать AI-анализ конкретного поставщика через backend и Ollama `gemma3:4b`.

## Формат реализации

Frontend сделан на React + TypeScript + Tailwind. Данные, фильтры, рейтинг и
AI-анализ приходят из FastAPI backend. Такой формат выбран, потому что он ближе
к реальному внутреннему инструменту отдела закупок: пользователь быстро
фильтрует список, сравнивает условия и запускает AI-рекомендацию только для
интересных поставщиков.

## Быстрый старт

```bash
cd frontend
printf "VITE_API_URL=http://localhost:8001\n" > .env
npm install
npm run dev
```
